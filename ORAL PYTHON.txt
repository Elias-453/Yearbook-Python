# 💻 Point Technique 2 : Algorithme de Floutage

## box_blur() - Protection de la vie privée

**Principe du Box Blur :**

Pour chaque pixel de l'image :
1. Sélectionner un carré de pixels voisins (rayon = 4)
2. Calculer la moyenne RGB de ces pixels
3. Remplacer le pixel central par cette moyenne

**Implémentation :**
```python
def box_blur(image, rayon=3):
    for y in range(hauteur):
        for x in range(largeur):
            total_r = total_g = total_b = 0
            compteur = 0
            
            # Parcourir les voisins
            for dy in range(-rayon, rayon + 1):
                for dx in range(-rayon, rayon + 1):
                    nx, ny = x + dx, y + dy
                    if 0 <= nx < largeur and 0 <= ny < hauteur:
                        r, g, b = pixels[nx, ny]
                        total_r += r
                        total_g += g
                        total_b += b
                        compteur += 1
            
            # Appliquer la moyenne
            pixels_resultat[x, y] = (
                total_r // compteur,
                total_g // compteur,
                total_b // compteur
            )
```

**Complexité :** O(largeur × hauteur × rayon²)

# 💻 Point Technique 3 : Génération (suite)

## Placement d'un élève sur la grille

**Calcul de position :**
```python
for i in range(début, fin):
    # Position dans la grille
    colonne = (i - début) % colonnes
    rangée = (i - début) // colonnes
    
    # Coordonnées pixel
    x = départ_x + colonne × (largeur_img + espacement)
    y = départ_y + rangée × (hauteur_img + espacement)
```

**Traitement de l'élève :**
```python
# 1. Charger et redimensionner l'image
img = Image.open(images[i]).resize((largeur_img, hauteur_img))

# 2. Récupérer les infos par ID
id_image = os.path.splitext(os.path.basename(images[i]))[0]
informations = eleves.get(id_image)

# 3. Appliquer le floutage si demandé

cadre_noir_image = ImageOps.expand(img, border=bordure, fill="black")
```

# 💻 Point Technique 4 : Centrage du Texte

## Problème : Texte décalé à gauche

**Avant (texte à gauche) :**
```python
draw.text((x, y), nom_complet, font=font)
```

**Solution : Calcul de la position centrée**
```python
# 1. Mesurer la largeur du texte
largeur_nom = draw.textlength(nom_complet, font=font)

# 2. Calculer le décalage pour centrer
position_x_centrée = x + (largeur_img - largeur_nom) // 2

# 3. Afficher le texte centré
draw.text((position_x_centrée, y + hauteur_img + marge_texte), 
          nom_complet, fill="black", font=font)
```

**Pour les citations multi-lignes :**
```python
def ecrire_texte_multiligne(draw, texte, position, font, largeur_max):
    # Découpe en lignes + centre chaque ligne individuellement
    for ligne in lignes:
        largeur_ligne = draw.textlength(ligne, font=font)
        x_centre_ligne = x + (largeur_max - largeur_ligne) // 2
        draw.text((x_centre_ligne, y), ligne, font=font)
        y += hauteur_ligne + espacement
```

# 💻 Point Technique 6 : Redimensionnement Adaptatif

## Deux stratégies selon le besoin

**Cas 1 : Photos élèves (taille fixe)**
```python
img.resize((largeur_img, hauteur_img))
```
→ Redimensionnement direct, peut déformer légèrement

**Cas 2 : Logo/Photo classe (proportionnel)**
```python
# Calculer le ratio pour remplir la page
ratio = max(largeur_cible / img.width, hauteur_cible / img.height)

# Nouvelles dimensions proportionnelles
nouvelle_largeur = int(img.width × ratio)
nouvelle_hauteur = int(img.height × ratio)

# Redimensionner avec interpolation BICUBIC (qualité)
img = img.resize((nouvelle_largeur, nouvelle_hauteur), Image.BICUBIC)
```

**Pourquoi BICUBIC ?**
- Interpolation cubique entre pixels
- Qualité supérieure (vs NEAREST ou BILINEAR)
- Évite l'effet pixelisé

# 🚧 Difficultés Rencontrées

## Défi 1 : Association ID ↔ Citation

**Problème initial :**
```python
citations = []  # Liste simple
for path in citations_files.values():
    citations.append(lire_fichier(path))

citation = citations[i]  # ❌ Ordre des fichiers ≠ ordre des images
```

**Solution :**
```python
citations_dict = {}  # Dictionnaire par ID
for id_eleve, path in citations_files.items():
    citations_dict[id_eleve] = lire_fichier(path)

citation = citations_dict.get(id_image)  # ✅ Accès par ID
```

# 🚧 Difficultés Rencontrées (suite)

## Défi 2 : Centrage du texte

**Problème :** Pas de méthode native `draw.text_centered()`

**Solution :** Calcul manuel de la largeur avec `textlength()` puis décalage

## Défi 3 : Performance du floutage

**Problème :** 3 boucles imbriquées → lent sur grandes images

**Solution :** 
- Limiter le rayon à 4 (bon compromis qualité/vitesse)
- Possibilité d'utiliser `ImageFilter.BoxBlur()` (natif, plus rapide)

## Défi 4 : Gestion des erreurs

**Problème :** Image manquante, citation absente...

**Solution :**
```python
try:
    img = Image.open(images[i])
except:
    print(f"Image introuvable : {images[i]}")
    continue  # Passe à l'élève suivant
```
