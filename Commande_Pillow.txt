box_blur(image, rayon)

Son Rôle est d'Appliquer un floutage aux images décrit par le dictionnaire .
ca detecte chaque pixel de l’image.
puis calcule la moyenne de la couleur des pixels voisins.
Et ca remplace le pixel original par cette moyenne.

- image: L'image PIL à flouter
- rayon : Intensité du flou 

Algorithme de floutage :
 Parcourt chaque pixel de l'image
 Pour chaque pixel, calcule la moyenne des pixels voisins dans un carré de taille (2×rayon+1)
 Remplace le pixel par cette moyenne
 Répète pour les 3 canaux RGB

Le redimensionnement de mes images:


Pour ajuster les images d’élèves dans le yearbook, avec :

img = Image.open(images[i]).resize((largeur_img, hauteur_img))


Pour adapter le logo ou la photo de classe à la taille de la page, avec un ratio dynamique :

ratio = max(largeur / img.width, hauteur / img.height)
nouvelle_taille = (int(img.width * ratio), int(img.height * ratio))
img = img.resize(nouvelle_taille, Image.BICUBIC)


Étape 1 : ouvrir l’image

Image.open(path) charge l’image depuis le disque et enregistre ses dimensions (largeur/hauteur).

Étape 2 : calcul du ratio de redimensionnement

Le ratio est calculé pour que l’image remplisse bien la page ou la zone attribuée :

ratio = max(largeur / img.width, hauteur / img.height)


Si l’image est trop large ,elle est réduite.

Si elle est trop petite  ,elle est agrandie.

Ca garde les proportions pour ne pas la déformer.

Étape 3 : création d’une nouvelle taille

nouvelle_taille = (int(img.width * ratio), int(img.height * ratio))


multiplie les dimensions d’origine par le ratio calculé.

Étape 4 : redimensionnement réel
img = img.resize(nouvelle_taille, Image.BICUBIC)


Le mode BICUBIC applique une interpolation douce entre les pixels, pour obtenir un bonne qualité d image 
Cela calcule une moyenne  des pixels voisins :
résultat plus fluide


Pour  box_blur détail des calcules : 

def box_blur(image, rayon=3):
    pixels = image.load()
    largeur, hauteur = image.size
    resultat = Image.new("RGB", (largeur, hauteur))
    pixels_resultat = resultat.load()

    for y in range(hauteur):
        for x in range(largeur):
            total_r = total_g = total_b = 0
            compteur = 0
            for dy in range(-rayon, rayon + 1):
                for dx in range(-rayon, rayon + 1):
                    nx, ny = x + dx, y + dy
                    if 0 <= nx < largeur and 0 <= ny < hauteur:
                        r, g, b = pixels[nx, ny]
                        total_r += r
                        total_g += g
                        total_b += b
                        compteur += 1
            pixels_resultat[x, y] = (
                total_r // compteur,
                total_g // compteur,
                total_b // compteur
            )
    return resultat

Étape 1 : récupérer les pixels
-pixels = image.load() permet d’accéder à chaque pixel individuellement

Étape 2 : parcourir tous les pixels

-Les deux boucles for y in range(hauteur) et for x in range(largeur) cherche les pixel un par un .
Pour chaque pixel (x, y), on prend tous les pixels voisins dans un carré de côté 2*rayon + 1

Étape 4 : calculer la moyenne des couleurs

On additionne les composantes R, G, B (couleur) de tous les pixels voisins :

total_r += r
total_g += g
total_b += b
compteur += 1
Puis on divise par compteur pour obtenir la couleur moyenne.

Le pixel central devient cette moyenne :
pixels_resultat[x, y] = (total_r // compteur, total_g // compteur, total_b // compteur)
ce qui donne un effet flou car chaque pixel devient la moyenne de ses voisins.
